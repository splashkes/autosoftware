# Validation

Manual validation for this realization should cover:

1. run the app from `artifacts/notepad-app/`
2. open the app in two browser tabs
3. create a note in one tab and confirm it appears in the other after refresh
   polling
4. edit the note and confirm the updated text appears
5. delete the note and confirm it disappears in both tabs

Automated validation can be added later once the realization boot contract is
settled.
