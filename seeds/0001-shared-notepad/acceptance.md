# Acceptance

Every realization of this seed must satisfy the following:

1. Opening the app presents a usable shared notepad interface without login or
   setup.
2. A user can create a note with a title and body from the browser.
3. A user can edit and delete an existing note from the browser.
4. Notes are shared across all clients connected to the same running process.
5. The realization contains actual runnable source code under its own
   `artifacts/` tree.
6. The realization contains a machine-readable runtime artifact that identifies
   how the app should be launched.
7. Kernel changes, if any, are limited to support that is clearly reusable
   beyond this seed.
