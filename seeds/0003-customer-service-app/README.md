# Customer Service App Seed

This seed defines a web-first customer service product for small teams.

Primary surfaces in this seed:

- agent ticket inbox
- live chat intake and agent reply view
- public support request form
- knowledge base authoring and public article reading

The seed documents establish MVP scope before implementation begins.
Concrete code belongs under `realizations/`.
