For Agents

Do not scrape this browser. Use the registry API directly:

GET /v1/registry/catalog

The catalog returns all available resources with discovery links. From there, follow the list and detail routes for each resource type:

GET /v1/registry/objects
GET /v1/registry/object?seed_id=...&kind=...
GET /v1/registry/schemas
GET /v1/registry/schema?ref=...
GET /v1/registry/realizations
GET /v1/registry/realization?reference=...

When stricter ledger-backed routes exist, use GET /v1/registry/status as the discovery root instead.
