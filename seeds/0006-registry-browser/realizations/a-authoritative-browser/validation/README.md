# Validation

Validation evidence for this realization should cover:

- no mutation controls in the browser
- no mutating HTTP verbs accepted by browser-facing registry routes
- browse coverage for objects, claims, schemas, change sets, and rows
- deep detail coverage for provenance, supersession, and schema-version links
- agent guidance that points to authoritative API routes rather than HTML

Seed-level requirements remain in `../../acceptance.md`.
