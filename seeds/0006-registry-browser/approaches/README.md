# Approaches

`approaches/` holds named, machine-readable approaches for the registry
browser seed.

Current approaches:

- `a-authoritative-browser` for a direct, trust-first catalog browser
- `a-ledger-reading-room` for a human-first refinement with stronger
  information architecture and clearer conceptual grouping

Each realization should point back to one approach through
`realization.yaml:approach_id`.
