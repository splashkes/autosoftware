# Webpresence Assets

This directory preserves standalone homepage directions that are adjacent to,
but not identical with, the kernel bootloader or any realized app surface.

Current preserved bundle:

- `as-standalone-sprout-homepage.tar`

Provenance:

- Derived from the earlier standalone sprout page on branch
  `codex/labs/notepad`
- Visual structure and sprout branding preserved from the retired standalone
  prototype that predated the current bootloader surfaces.

Intent:

- Keep a closer copy of the old three-option standalone homepage for future web
  presence work.
- Preserve it separately from the kernel runtime boot surface so future agents
  can inspect and revive the design without digging through git history.
