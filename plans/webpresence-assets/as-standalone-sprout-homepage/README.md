# AS Standalone Sprout Homepage

This bundle preserves a closer copy of the earlier standalone AS homepage with
the three mocked seed options:

- Customer Service App
- Event Listings
- Charity Auction Manager

It is intentionally static and self-contained.

Notes:

- `index.html` is a static adaptation of the earlier mock boot page.
- `assets/sprout-logo.css` and `assets/sprout-logo.js` preserve the sprout
  animation treatment.
- The interactive controls are local-only and do not call any backend.
